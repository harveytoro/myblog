Apps in my Launchpad

I have about 10 apps that I use all the time, since getting my macbook I have only installed about 20 apps which saves cluttering it up with stuff I would only use about once. So in this blog post I am going to name the 10 apps that I work with on a day to day bases.

1. Sublime Text 2

Sublime text 2 is a clean uncluttered code editor, what makes it so good is that it just lets you code, but in the background it has powerful features. By installing packages you can increase the features.

2. CodeRunner

Whether you are learning a new language or want to write and run an algorithm quickly CodeRunner is the app to use, it allows you to run code on a inbuilt console in multiple different languages.

3. FileZilla

Is a powerful FTP client, if you have to use FTP then I suggest you use FileZilla. Alternatively I suggest you push your files to a private repo and use a service like FTPLOY to FTP it to your host.

4. GitHub

Probably a good time to mention the Github app which I use to push code to Github. (obviously)

5. Markdown Pro

Markdown pro is a brilliant markdown editor which I am currently using to write this blog post and use to write all of my university notes. You can export to HTML or PDF, and lets face it markdown is pretty cool you don't have to use a clunky word processor, just write markdown in fact everybody should be using it.

Used to write Java code for my uni course work.

We all know what pocket does, so will save my typing.

10. Google Chrome

I have tried most web browsers and I did get on well with safari, until I started using the 'Inspect Element' feature so I changed back to chrome.
